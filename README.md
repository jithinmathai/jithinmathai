## 🚀 About Me

I design and ship **cloud-native systems and AI agent orchestration platforms**. By day I build Spring Boot microservices that go native with GraalVM; by night I'm wiring LLMs into multi-agent workflows with LangGraph, AutoGen, and CrewAI. I care about systems that are **fast, observable, and small enough to fit in a container someone actually wants to run**.

- 🤖 **Agent orchestration** — supervisor/worker hierarchies, tool calling, memory, RAG, evals & HITL.
- ☕ **Backend + GraalVM** — Spring Boot 3, native images, < 100ms cold starts.
- ☁️ **Cloud-native** — Azure / AWS / GCP, Kubernetes, event-driven microservices.
- 🔌 **Protocols I'm bullish on** — **MCP**, **A2A**, function calling, OpenAPI-as-contract.
- 💬 **Ask me about** — LangGraph, Spring AI, GraalVM, Azure architecture, native-image gotchas.
